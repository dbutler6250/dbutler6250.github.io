Force Indentation Machine

This project is one of my most involved. I have provided support for this device for around two and a half years while in school. The main application for this project was originally as a force-indentation machine. It is dynamic enough to be applied to other kinds of surface science, like surface wetting which is what it has been used as for some time now. The machine now utilizes a captive nut stepper motor to move with sub-micron precision! It utilizes a reflection based optical sensor to obtain data on the position of the acrylic and other assorted cantilever types. This motion control and position sensing capability allows this machine to be used for a variety of applications.

The main attraction for this machine is the Java based GUI that I created. The software provides a flexible user interface to log sensor data, change the sensor parameters, control the stepper motor, and run programmed experiments. This Java application is what I have supported for such a long time and has taught me so much about interfacing with sensors and the user at very high speeds, reliably.
